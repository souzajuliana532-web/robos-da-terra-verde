# 🤖 A Grande Missão dos Robôs da Terra Verde

Um site interativo contando a história dos robôs agrícolas na Fazenda Terra Verde!

## 📖 Sinopse

Nesta aventura especial, acompanhe o Fazendeiro Beto e seus três robôs ajudantes:
- **TICO**: Um robô com rodinhas que remove plantas daninhas
- **BRAÇO-MECÂNICO**: Um robô com braço longo para colheita de frutas
- **ZECA**: Um drone que monitora a fazenda do céu

Todos trabalhando juntos para manter a Fazenda Terra Verde produtiva e feliz!

## ✨ Características

✨ **Animações Fluidas**: Cada cena conta com animações CSS criativas
🔊 **Efeitos Sonoros**: Sons sintetizados em tempo real para cada ação
📱 **Responsivo**: Funciona em desktop, tablet e mobile
⌨️ **Navegação**: Use botões, setas do teclado ou clique nas cenas
🎬 **5 Cenas Principais**: Cada uma com sua própria história e interatividade

## 🚀 Como Usar

1. **Abra o arquivo** `index.html` em seu navegador
2. **Clique nos botões de navegação** para trocar de cena
3. **Pressione as setas** ← → para navegação rápida
4. **Clique em "Reproduzir Cena"** para ver animações e ouvir efeitos sonoros

## 🛠️ Tecnologias

- **HTML5**: Estrutura semântica
- **CSS3**: Animações e design responsivo
- **JavaScript Puro**: Gerenciamento de cenas e Web Audio API para sons
- **Sem dependências externas**: Tudo funciona offline!

## 🎵 Efeitos Sonoros

Os sons são gerados sinteticamente usando a Web Audio API:
- **BIP/BLIP**: Sons de inicialização dos robôs
- **VRUM**: Motor do robô Tico
- **PLOC**: Planta sendo arrancada
- **ZIIIP**: Braço mecânico esticando
- **FOF**: Fruta na cesta
- **ZUUUM**: Drone voando
- **CLIQUE**: Câmera do drone fotografando
- **CHUá**: Água molhando plantação
- **PIM/POU**: Robôs desligando

## 🎨 Personagens

### Fazendeiro Beto
O corajoso fazendeiro que confia em tecnologia para melhorar sua plantação!

### Robô Tico
Com suas rodinhas fortes e pinças precisas, Tico remove as plantas daninhas que prejudicam o crescimento.

### Robô Braço-Mecânico
Com seu braço longo e precisão, colhe frutas delicadas sem amassar.

### Drone Zeca
Voa alto no céu monitorando toda a fazenda com sua câmera inteligente.

## 💡 Dicas de Uso

- Abra o navegador DevTools (F12) para ver o console sem erros
- Os sons funcionam melhor com volume ativado
- Teste em diferentes tamanhos de tela para ver a responsividade
- Experimente clicar várias vezes no botão "Reproduzir Cena"
